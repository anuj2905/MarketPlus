# MarketPlus
MarketPlus is a fast and user-friendly stock trading platform inspired by Zerodha. It lets users buy and sell stocks, track live market prices, create watchlists, view charts, and manage portfolios securely. The platform focuses on simple design, quick order execution, and a smooth trading experience.

MarketPlus is a simple and fast online stock trading platform inspired by Zerodha. It allows users to buy and sell stocks, track live prices, manage portfolios, and view market charts in real time. The platform focuses on speed, clean UI, and a smooth trading experience for beginners and experienced traders.

Features

User signup & secure login

Buy & sell stocks instantly

Live market price updates

Watchlist creation

Portfolio management

Order history tracking

Interactive charts & market analytics  

Fast and responsive UI

Tech Stack

Frontend: HTML, CSS, JavaScript / React (optional)

Backend: Node.js / Express

Database: MongoDB

Authentication: JWT / Passport

Charts: TradingView / Chart.js

Deployment: GitHub & AWS
